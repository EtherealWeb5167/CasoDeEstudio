# Caso de Estudio: arboles y Grafos para Innovatec


## Características Implementadas

### Parte A: Árbol - Jerarquía Organizativa

Construcción de Árbol: Permite añadir empleados y departamentos de forma jerárquica.
Funciones Principales:
Insertar: Añadir nuevos nodos a la estructura.
Buscar: Encontrar un empleado o departamento específico.
Contar: Obtener el número total de nodos en el árbol.
Profundidad: Calcular los niveles de la jerarquía.
Recorridos Clásicos: Implementación de los recorridos Pre-Orden, Post-Orden e In-Orden para demostrar el manejo de la estructura.

### Parte B: Grafo - Sistema de Rutas del Parque

Representación del Grafo: Se utiliza una Matriz de Adyacencia para modelar los edificios (vértices) y las rutas (aristas ponderadas).
Funciones Principales:

Añadir nuevos edificios al parque.
Crear conexiones (rutas) entre edificios con una distancia específica.
Algoritmo de Ruta Más Corta: Implementación del Algoritmo de Dijkstra para encontrar el camino más eficiente entre dos edificios.
